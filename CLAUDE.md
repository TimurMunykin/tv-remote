# AGENTS.md

## Правила для AI-агентов

### Docker — единственный способ запуска
- Всё работает через Docker. Ничего не устанавливается локально, только через Dоcker. Всякие проверки теорий и прочего - тоже только через Docker
- Запрещено устанавливать что-либо на сервере напрямую (`apt`, `pip`, `npm` и т.д.).
- Новые зависимости добавляются только через `requirements.txt` и `Dockerfile`.

### Деплой — только через rebuild.sh
- После **любых** изменений деплой выполняется **только** командой `./rebuild.sh`.
- Не использовать `docker compose up --build` или `docker restart` напрямую.
- Скрипт сам определяет нужна ли пересборка образа или достаточно перезапуска.
